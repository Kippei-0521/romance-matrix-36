# 恋愛における隠れた性格診断 | 36-Type Romance Personality Matrix

![Project Banner](https://via.placeholder.com/1200x600/fff5f7/ff85a1?text=Romance+Personality+Matrix)

**恋愛における隠れた性格診断** は、深層心理学に基づいた「価値軸 × 行動軸」の36タイプ診断サイトです。
可愛らしく洗練されたUIと、"怖いくらい当たる"詳細な分析結果を提供します。

## ✨ 特徴

- **36タイプの精密診断**: 6つの深層価値観（X軸）と6つの恋愛スタイル（Y軸）を掛け合わせた36通りのマトリックス。
- **心理学的な詳細レポート**: 「魂の渇望」「Shadow Analysis（陥りやすい罠）」など、踏み込んだ心理分析を提供。
- **キャラクターイラスト**: 診断結果に合わせた独自キャラクターを表示。
- **相性診断**: ベストマッチと注意すべきタイプを算出し、具体的なアドバイスを提供。
- **直感的なUI**: 5段階のリッカート尺度を用いた、答えやすく視覚的に美しい回答システム。

## 🛠 技術スタック

- **Frontend**: Next.js (App Router), React, TypeScript
- **Styling**: Tailwind CSS v4, Framer Motion (Animations)
- **Icons**: Lucide React
- **Hosting**: Vercel (Recommended)

## 🚀 開発の始め方

### インストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。
